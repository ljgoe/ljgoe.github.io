This describes how to build a MTR for lab purposes.

Prerequisites:

  • Parallels Desktop or other choice of virtual machine hosting software
  • Windows Virtual Machine running windows 11
  • Office 365 tennant with administration rights
  • Microsoft teams Pro or Basic licence
  • Access to the windows 11 Image with these exact specs (here is an example)
  Major release: 22H2 (Win 11)
     Name in VLSC: Windows 11, version 22H2 (updated Sept 2022) x64
     Size (bytes): 5440065536
     SHA256: 768A49D9DDB9997E427094D317329E113E6ABE74DBB5707B4AAEB8270ADC6925

Installation Steps

  • Install Windows 11
  • Download the CreateSrsMedia.ps1 script.
  • Run powershell as Admin
  • Allow running scripts, execute command
powershell -executionpolicy unrestricted

Build Virtual Machine

Once the scripts have run you will have the media required to build the MTR.

Create new virtual machine and build it from the install file which was created in the previous steps.

At the final stage, you will see the setup page of your newly installed virtual MTR. You can accept the terms and configure your account, but will end up at this screen which will not allow you to continue.

Exit this setup, by going back to the EULA screen and press Exit.

Login to New MTR

At the user screen, login as the local administrator.

  • User: administrator
  • Password: “sfb” (don’t forget to change this!)

Browse to the following folder:

C:\Users\Skype\AppData\Local\Packages\Microsoft.SkypeRoomSystem_8wekyb3d8bbwe\LocalState

Create a new SkypeSettings.xml-file with the following settings:

    <SkypeSettings>
    	<UserAccount>
    		<SkypeSignInAddress>test@test.com</SkypeSignInAddress>
    		<ExchangeAddress>test@test.com</ExchangeAddress>
    		<Password>test</Password>
    	</UserAccount>
    </SkypeSettings>

Reboot the MTR and it will use the XML file to configure the account. Thats it, the device should be registered and ready to test.

After a software update the new Teams UI should show

Teams Room Pro Portal

The device should now also show in the Teams Room Pro Management Portal if you have the teams room pro licence

Click on the device name and enrol it
